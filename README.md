# Lumina | Architecting Legacies

Welcome to the **Lumina** repository, a premium, full-stack luxury real estate showcase application built with Next.js and Tailwind CSS. 

## 🌟 Overview

Lumina is designed to deliver a high-end, immersive web experience. It features sophisticated micro-animations, smooth scrolling, magnetic hover interactions, and a clean, brutalist-inspired luxury aesthetic.

## 🚀 Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion (3D tilt, scroll reveals, stagger effects)
- **Scrolling:** Lenis (Smooth momentum scrolling)
- **Language:** TypeScript

## 📂 Project Structure

- `/src/app` - Next.js application routes and global styles (`globals.css`).
- `/src/components` - Modular React components (Navbar, Hero, BentoGrid, Philosophy, Testimonials, Footer).
- `/public/images` - Static image assets optimized for high-resolution displays.

## 💻 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application in action.

## 🎨 Design System

The application utilizes a curated, harmonious color palette defined in `globals.css`:
- **Surface/Background:** `#050505` (Deep Black)
- **Primary:** `#d4af37` (Luxurious Gold)
- **On-Surface Variant:** `#d0c5af` (Sand/Beige text for optimal contrast)

All fonts are dynamically loaded via `next/font/google` (Playfair Display for headings, Inter for body text).
